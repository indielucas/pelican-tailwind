# installation
```
pip install pelican
npm init -y
npm i -D tailwindcss
npm i -D @tailwindcss/typography
npm i -D daisyui@latest
```

# development
```
npm run watch:css
```

# build style
```
npm run build:css
```

# build content
```
pelican content
```

# preview your site
```
pelican -r -l
```
